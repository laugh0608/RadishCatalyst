# 从0.5开始大型游戏开发

## 第0节 为什么是 0.5

游戏开发, 四个字里面融汇了数学、美术、程序、文学、音乐、金融等多个领域

它不同于开发一个BS系统, 经理、码农、UI再给你配个数据库顶天了, 它是一个非常庞大的系统性工程

本文希望能通过通俗易懂的方式带各位佬u上手游戏开发, 而不是去开发游戏引擎、绘制精美角色等轮子工作, 还是用开发 B/S 举例, 你不能指望一个人从数据库驱动这种底层轮子开始能在这个人兴(耐)趣(心)耗尽之前完成整个系统的开发

那么在游戏开发中有哪些属于轮子呢?

* 游戏引擎
* 美术资源
* 动画
* (底层)玩法设计
* 音乐
* 联机库
* 硬件适配库

这里不得不指出一个很大的误区, 有不少佬u试图用AI去包揽美术和音乐??? 没必要, 真的

游戏的美术资源需要光影贴图、纹理贴图、骨架这种拆分非常细的素材, 2D 游戏是看上去2D实际上并不是 2D 的, AI 生成的纯 2D 图片用在游戏中会有大量不适配需要手动拆分调整的地方

而音乐可能很多佬u没有留意到, 游戏音乐是一种即时音乐, 即为可以随时根据场景、氛围、剧情无感的从任意时间点开始播放, 你不管从 OST 那一节开始播放都不会觉得突兀违和, 而且可以连着无限循环而不会让玩家察觉

然后玩法设计, 特指底层玩法设计是很多新手(包括我)踩坑重灾区, 千万不要在玩法设计上有奇思妙想, 因为你这样很容易撞到空白领域, 出现没有现成的轮子需要自己造的情况, 自己造玩法如果没有设计好那你游戏从根就烂掉了, 最好不要冒这个险

## 第3节 书写一段故事

不管什么游戏都是在讲述一个故事, 哪怕是开心消消乐这种游戏都有一个背景故事存在, 它的目的是串联游戏的的一切, 让游戏设定合理化, 提高玩家的代入感

一个好的故事, 例如《黑神话：悟空》, 大家看到名字看到零碎剧情就会有玩的欲望, 但我要是换成《猴王历险记》然后告诉你就是一只猴子一路打怪升级, 你的游玩欲望会强吗?

游戏中很多与玩法半毛钱关系都没有的东西是专门为叙事设计的, 还是用《黑神话：悟空》举例, 开场那个打斗过场砍掉能节省大量经费, 那我看到改成一段文字或者拼插式漫画你觉得还能吸引你吗?

这个故事的作用一是吸引你进来玩, 二是从乏味的流程中吊你胃口让你能继续玩下去

一个设计时长十几小时几十小时的游戏, 我再怎么牛的玩法设计也会枯燥的

游戏是第九艺术, 艺术之所以能成为艺术是因为它承载了思想

反战、拼搏、左倾、右倾、批判吧啦吧啦的, 里面掺满了开发者(组)的意(私)志(货), 把游戏做出来然后发布付出了这么多要么是为了获得认可要么是为了收割, 它和艺术应该是完全不沾边的

但艺术就是这样, 艺术没有好坏之分, 全世界几十亿人只要有1个人能懂, 那它此刻就已成艺术

而天马行空的玩法、精致的美术、动听的音乐单独是无法承载思想的, 只有其中讲述的故事才能, 希望能点醒各位已经开始尝试游戏开发但还迷茫的佬u

想想你有什么深刻的东西想和其他人讲述, 将它写成一段故事, 把它做成游戏, 让它推动你的开发

## 第4节 设计一个玩法

你首先需要从中设计出核心玩法循环, 通过核心玩法循环螺旋上升带动游戏整体发展, 例如“探索→战斗→升级→探索”这就是一个玩法循环

然后你需要思考操作(或者说是交互)方式, 是键盘、鼠标点击、拖动、滑动?

你的游戏如何压制发育节奏? 通过经济系统? 那你还需要设计货币获取/消耗、交易系统、物价平衡等等

你说经济系统太复杂了打算通过DND那种角色成长系统? 那你也得设计成长线、技能树、装备系统、等级机制等等

如果涉及战斗还需要设计战斗机制、伤害计算等等

如果还有联机功能, 那你是不是要考虑合作还是对抗? 要不要公会、好友、聊天?

想搞点能延长游戏寿命的? 随机事件、多结局、Roguelike了解一下?

哦, 还有别忘记新手引导, 永远不要高估用户的智商

这些都属于玩法设计

## 第5节 开始你的叙事

哎哎哎, 别急着开发嗷

故事写好了, 玩法也设计好了, 你还缺叙事设计哦,

从主线、支线到对话、环境再到关卡和角色

这是一张错综复杂的网, 不要高估了你的组织能力, 不提前规划设计好, 一边写一边设计只会越来越来乱

一个大型游戏你最少游戏时长得有10个小时吧?

那你除了主线就必须有支线, 支线的作用有两个, 一是拖时长, 二是调节情绪, 比如一个主线压抑的游戏可以在支线加点温情来避免玩家情绪持续走低

主线有了支线也有了, 你打算如何体现剧情呢? 别想复杂了, 就用对话与剧本, 通过角色对话、过场动画脚本和环境来叙事(别整一大段几万字的文本给用户看啊!

然后为了配合叙事节奏, 你需要通过任务或者关卡来引导玩家, 同时营造紧张→舒缓→高潮的情绪释放点来调动多巴胺

最后就是搭配主线、支线、任务和关卡的 NPC 了, 这个我不多说, 因为很难, 我也说不准

一个好的 NPC 设计, 怎么说呢… 2077里面那个赖宣? 这个真得自己体会, 我不知道如何用文字去表达这样 NPC, 他有血有肉就像真人一样, 但这是如何做到的, 我说不出来

反正就是角色的外观、性格、背景故事、能力设定、角色之间的关系、阵营、互动等等组成了一个角色的一生
