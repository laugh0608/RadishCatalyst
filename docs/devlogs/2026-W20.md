- 井纹架断面包和井系桥断面包继续复用污染过滤器和基础反应器，不引入第三台设备、电网、管线或自动物流。

- 回传锚点交互改成“多已部署落点 + 单当前回投目标”：保留所有已校准锚点，但基地回投仍只回到当前选中的那个，不扩成自由传送网。
- `S12` GUI 人工复测通过后，当前阶段默认先做方向校准，不把“相位井锚核之后”直接实现成下一个同构门禁包。
- `docs/design/onboarding-and-first-hour.md` 收回真正首小时范围；`S6` 之后的深段连续主线改由 `current.md`、`vertical-slice.md`、开发复测基线和周志维护。
- 锚核后的下一包明确改成“前线短守场 + 局部稳定窗口”结构：仍复用基础反应器和污染过滤器，但把外勤目标从回收两处材料切换为带着基地产物到现场完成守场。
- 锚场回稳失败后不强制玩家回基地重做 `井系校锚桩`；已部署桩保留在现场，允许直接重试，避免把守场失败又退回成一轮重复采集。

- 文档篇幅治理按角色分级，而不是对全部 Markdown 套一个统一硬上限：入口文档立即强制，活跃专题文档、周志和参考文档先用 warning 推进渐进拆分，避免一次性把存量长文全部打成错误。
- `S0` 通过后，下一阶段改为「资源循环与前线节奏重构」：不继续堆相位井后段线性包，先治理基础零件稳定供给、后段任务模板重复和无意义往返基地成本。
