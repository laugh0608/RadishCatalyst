# RadishCatalyst / 异星催化 创作开发准备总纲 - 定位与原则

返回：[RadishCatalyst / 异星催化 创作开发准备总纲](creative-development-brief.md)

## 1. 文档目的

这份文档不是单纯的灵感汇总，也不是泛泛而谈的大纲，而是面向“可以准备开始创作、开发、规划”的开工总纲。

项目正式名称：

- 英文名：`RadishCatalyst`
- 中文名：`异星催化`

它基于 `docs` 目录下已有文档中的共识与灵感，重新收束出一个更明确的方向：

- 以化工自动化为独特卖点
- 以人物探索、战斗、升级、养成为主驱动
- 以弱联网到多人协作 ARPG 为中长期方向
- 以个人开发者可执行为首要约束

它的用途是：

- 为项目立项定调
- 为后续策划文档提供上位约束
- 为技术和开发顺序定边界
- 避免项目再次滑向“大而全但无法开工”

## 2. 项目一句话定义

这不是一个单纯的工厂游戏，也不是一个传统 MMO。

它更接近：

**“以异星化工基地为核心、生存探索为外层、人物成长与战斗为主线、后期支持联机协作的 2D / 2.5D 工业科幻 ARPG。”**

更具体一点：

玩家扮演一名登陆危险星球的工程战斗员，在建设自动化化工基地的同时，外出探索、采集、战斗、升级、解锁新工艺与装备，并逐步从单人生存扩展到多人协作开发、基地共建和组队远征。

## 3. 最终方向为什么这样定

这份总纲吸收了前面几类想法：

- 化工自动化：来自你最独特的专业优势
- 人物探索与成长：来自修仙项目里最强的长期驱动力
- ARPG 战斗：保证过程手感，不让玩法变成纯后台模拟
- 联机：保留长期扩展空间，但不在首版承担全部复杂度

这个方向的价值在于：

- 它比纯修仙 MMO 更容易做出差异化
- 它比纯工厂游戏更容易提供角色代入和持续目标
- 它比纯多人游戏更适合个人开发者先做出完整可玩版本
- 它允许你把“化工专业壁垒”变成真正的核心卖点

## 4. 汲取百家之长后的核心定位

这个项目不应简单模仿某一款作品，而应明确吸收不同类型作品的长处。

### 4.1 从自动化建造类里吸收什么

- 自动化链路扩张的成就感
- 物流与流体系统带来的空间规划乐趣
- 科技树逐步解锁更复杂流程的成长感
- 从低级生产到高级工业闭环的长期目标

### 4.2 从 ARPG 里吸收什么

- 角色操控与即时战斗的手感
- 野外探索和风险回报关系
- 装备、技能、属性、Build 的持续成长
- Boss、精英怪、远征区域带来的战斗目标

### 4.3 从养成经营类里吸收什么

- 基地是“家”，不是纯数值后台
- 玩家回基地时应该有明显恢复、制造、整理、规划的节奏切换
- 生产系统不只是赚钱，而是服务于角色成长和探索推进

### 4.4 从 MMO-like 设计里吸收什么

- 在线感
- 协作目标
- 基地共建或组队探索
- 聊天、好友、公会式组织成长

但不直接照搬：

- 大规模世界同步
- 重运营经济系统
- 首发即多人主城万人同屏

## 5. 核心幻想与玩家体验

玩家应该感受到的不是“我在摆机器”，而是：

- 我在一颗危险星球上建立自己的工业前哨
- 我在亲自外出探索、战斗、回收资源、解锁禁区
- 我建设的化工设施在支撑我的装备、弹药、药剂、能源和生存能力
- 我的人物成长与基地成长是相互咬合的
- 后期我不是一个人在生存，而是在和其他玩家协作征服环境

项目需要同时满足四种体验：

- 工程感：流程、设备、效率、工艺优化
- 冒险感：外出、战斗、地图推进、未知威胁
- 成长感：角色等级、装备、技能、科技同步变强
- 归属感：基地不断扩张，逐渐成为真正的据点

## 6. 最核心的设计原则

### 6.1 基地服务于冒险，冒险反哺基地

不能把工厂系统和人物系统做成两张皮。

正确关系应该是：

- 外出探索带回稀缺资源、样本、遗物、蓝图
- 基地把这些资源转化为装备、药剂、模块、能源、建材
- 新装备和新能力让玩家能进入更危险的区域
- 更危险的区域又要求更高水平的自动化和后勤支持

### 6.2 化工自动化是特色，不是门槛

不要把玩法做成专业考试。

应该做到：

- 看得懂
- 玩得出差异
- 优化有回报
- 失误有代价

不要做到：

- 让玩家手算化工公式
- 让系统过早复杂到劝退
- 让每个设备都像真实软件仿真器

### 6.3 联机是扩展方向，不是首版依赖

首版必须能作为单机或本地伪联网版本成立。

换句话说：

- 没有多人，游戏也必须好玩
- 多人只是在后续放大资源协作、分工和远征体验

### 6.4 战斗必须是真正核心，而不是点缀

这个项目不是纯工厂游戏。

所以战斗不能只是：

- 出门打两下素材怪
- 用于拖时间的采集前置动作

而应该承担：

- 地图推进
- 区域解锁
- 资源获取门槛
- 装备成长验证
- 联机协作价值体现

## 7. 推荐的一句话题材设定

最稳的题材方向仍然是：

**异星化工殖民。**

推荐设定版本：

人类在一颗高腐蚀、高毒性、地表生态极不稳定的星球建立前哨站。玩家既是工程操作员，也是外勤战斗员。为了活下去并推进殖民计划，玩家必须建立一整套从采集、反应、分离、储运、供能到安全治理的工业体系，同时不断外出调查遗迹、清理威胁、回收技术，并最终扩展成可联机协作的大型工业据点网络。

这个设定的好处是：

- 自动化合理
- 战斗合理
- 探索合理
- 科技树合理
- 联机合理
