- 后段解析类配方已移除固定 `基础零件 x2` 消耗，回基地解析不再因为缺基础零件而转成补件往返。

## 今日事项

1. 用 `S8` 到 `S13` 基线复核连续整备包的体验：回基地应一次完成过滤中间件和装配件，下一趟外勤直接进入现场目标。
2. 继续观察装配类配方的基础零件压力，后续新增装配件优先考虑批量加工、现场目标或可选模块，不把解析、读数或坐标转换重新做成固定零件税。
3. 维护资源预算自动检查和可选补给门槛，后续新增配方、任务件消耗或阶段补给时同步更新预算口径。
4. 下一轮节奏重构优先补非同构现场目标，而不是继续把相位井线性包向东延长。
