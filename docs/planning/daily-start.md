## 明天事项

1. 补齐基础零件缺料提示和可选补给方向，让 HUD、设备面板或日志能指向前哨核心回收 / 阶段补给批次。
2. 维护资源预算自动检查，后续新增配方、任务件消耗或阶段补给时同步更新预算口径。
3. 重排“前线回传锚点”之后的后段节奏，减少同构任务和无意义往返；回基地应变成整备、选择、升级或批量加工。
4. 基于 2026-05-13 的四次提交做一次短复核，确认后续改动继续围绕资源循环与前线节奏重构，而不是继续横向堆相位井线性包。
