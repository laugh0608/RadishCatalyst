## 用途

当提示是“根据项目规划和开发进度，今天要来做什么以推进开发”时，优先阅读本文。本文只回答今天如何推进，不承载完整历史。

## 当前阶段一句话

当前处于「首小时引导可玩」阶段：验证玩家是否能在无口头解释下理解并完成“外出获得资源 -> 回基地加工 -> 远征产物改变下一次外勤结果”。

## 最近进度

- 回传上线后的下一包深段内容已继续推进到相位井锁：`S5` 现在会把玩家从基地相位回投台直接带到裂相碎屑、透镜胚片、中继调谐镜、裂相尖塔和内层故障轨迹这条链上，`S6` 则接着观察故障轨迹解析、故障残渣、故障芯、相位井钥和相位井锁。
- 恢复后的前哨核心现可作为基地整备点，回基地可直接恢复生命 / 防护，避免修复凝胶和旧图材料耗尽后形成软锁。
- 调试面板新增 GM 资源 / 状态调整区，可直接调整关键背包资源并一键回满生命 / 防护，方便 `S4` / `S5` 基线复测与深段内容开发。
- 开发复测基线 `S0` 到 `S6` 继续保留：`S4` 负责第二轮深段续跑，`S5` 负责基地回投与裂相尖塔段观察，`S6` 负责内层故障轨迹后的新闭环观察。

## 今日推荐推进

1. 默认先从 `S6` 起跑，完整走一遍“内层故障轨迹 -> 相位井坐标印片 -> 故障残渣 -> 故障芯 -> 相位井钥 -> 相位井锁 -> 相位井定位器”，确认回传后的新风险、收益和推进目标是否已经明确。
2. 若需要回看回投节奏，再退回 `S5` 观察“相位回投台 -> 裂相碎屑 -> 透镜胚片 -> 中继调谐镜 -> 裂相尖塔 -> 内层故障轨迹”；若只差关键资源或生命 / 防护，直接使用调试 HUD 的 GM 区补状态，不默认回头重打整条链。
3. 若这条新链稳定，下一轮主线就以相位井定位器为锚继续推进更东侧内层相位井；短期仍不新增第三台设备或复杂基地系统。
