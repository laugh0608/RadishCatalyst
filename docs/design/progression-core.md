# Character Progression And Equipment - 成长核心

返回：[Character Progression And Equipment](character-progression-and-equipment.md)

## 文档目的

这份文档用于定义 RadishCatalyst 的角色成长、装备、技能和 Build 在项目方向中的位置。

它不是数值表，也不是完整技能树方案。它先回答：

- 玩家为什么需要成长。
- 成长如何服务探索、战斗、基地和区域推进。
- 装备、技能、消耗品和基地生产如何咬合。
- 首版应保留哪些 RPG 深度，避免过早膨胀。

## 核心原则

角色成长必须服务主循环：

**基地生产提供装备、模块、药剂和特殊弹药，角色用这些能力进入更危险区域，带回更高价值样本和资源，再推动基地解锁更强生产能力。**

成长不能只来自刷怪经验，也不能只靠基地科技树自动变强。

玩家应该同时感受到：

- 自己的角色变强。
- 装备和工具更适配环境。
- 基地后勤让远征更稳。
- 新区域要求新的准备和 Build 取舍。

## 成长来源

### 外勤经验

来源：

- 战斗。
- 探索新地点。
- 扫描和采样。
- 完成区域目标。
- 秘境调查。

作用：

- 解锁角色基础能力。
- 提升武器、技能或外勤专精。
- 让玩家形成对某种冒险风格的投入。

外勤经验不应成为唯一成长路径。玩家只刷怪但不经营基地，不应长期推进高危区域。

### 基地工艺

来源：

- 样本分析。
- 新资源处理。
- 设备升级。
- 污染治理。
- 遗迹或秘境回收。

作用：

- 解锁装备材料。
- 解锁防护模块。
- 解锁药剂、特殊弹药和探测工具。
- 解锁区域通行条件。

基地工艺是本项目区别于传统 ARPG 的关键。它应让玩家意识到：战斗能力来自工业后勤，而不是只来自掉落装备。

### 区域适应

来源：

- 探索特定区域。
- 处理特定污染。
- 收集当地样本。
- 完成区域调查或治理。

作用：

- 获得针对环境的防护能力。
- 降低特定危险的远征成本。
- 解锁新的地图层、资源点或秘境入口。

区域适应可以作为轻量长期成长，但不应变成需要玩家在每个区域重复刷满的疲劳系统。

## 角色属性方向

首版属性应少而清晰，优先服务动作、风险和后勤判断。

推荐保留：

- 生命：承受直接伤害。
- 防护完整度：表示防护服、护盾或外勤装甲状态。
- 能量或体力：驱动闪避、工具、技能或短时防护。
- 负重或背包容量：限制远征收益和撤离判断。
- 环境抗性：对应污染、腐蚀、毒性、低温、高温或异常能量。

暂不建议做过多传统属性，例如大量力量、敏捷、智力、精神、暴击、命中、闪避和复杂抗性堆叠。首版重点应是“能不能安全进入、能不能打赢、能不能带回、值不值得继续深入”。
