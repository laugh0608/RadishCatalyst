# Onboarding And First Hour - 首小时流程

返回：[Onboarding And First Hour](onboarding-and-first-hour.md)

## 文档目的

这份文档用于定义 RadishCatalyst 首次游玩的前 30 到 60 分钟体验。

它不是完整教程脚本，也不是 UI 文案表。它只回答：

- 玩家第一次进入游戏时按什么顺序学习系统。
- 哪些概念必须先学，哪些延后。
- 如何避免玩家被化工、基地、战斗、地形和污染同时压垮。
- 首小时结束时玩家应理解什么。

## 首小时目标

首小时结束时，玩家应该理解：

- 我是谁：前哨工程战斗员。
- 我在哪：异常星球上的受损化工前哨。
- 我为什么出门：基地缺资源和样本。
- 我为什么回基地：资源需要加工，装备和补给来自基地。
- 我为什么要处理污染：污染影响探索、敌人和建造。
- 我为什么变强：基地工艺产出了装备、药剂、模块或通行能力。

如果玩家只记住一句话，应是：

**基地让我能走得更远，远征让我能把基地建得更强。**

## 教学原则

- 一次只教一个新系统。
- 每个新系统必须立刻让玩家获得反馈。
- 先做动作，再解释原因。
- 用任务目标引导，不用大段说明。
- 教程文本要短，优先使用 UI 高亮、地图标记和设备状态。
- 玩家失败后应知道如何补救。
- 当主线开始重复往返更深区域时，应优先压缩无决策空跑，而不是让玩家为“基地有价值”额外支付纯移动时间。
- 不在第一小时展示完整复杂度。

## 首小时节奏

### 0 到 5 分钟：醒来或抵达前哨

玩家体验：

- 看见基地平台。
- 获得移动控制。
- 接收到前哨受损提示。
- 看到一个明确目标：检查前哨核心。

教学内容：

- 移动。
- 镜头。
- 交互键。
- 任务目标显示。

避免：

- 立即打开完整基地 UI。
- 立即介绍全部资源和配方。
- 大段世界观说明。

### 5 到 10 分钟：恢复基础设备

玩家体验：

- 打开基础库存。
- 查看一个设备缺料或停机。
- 明确需要外出找晶体矿物。

教学内容：

- 库存。
- 设备状态。
- 缺料提示。
- 基地是后勤核心。

关键反馈：

- 设备面板显示缺少输入。
- 地图标出晶体矿脉方向。

### 10 到 20 分钟：第一次外出采集

玩家体验：

- 离开基地平台。
- 到达晶体矿脉区。
- 使用工具采集资源。
- 遭遇少量基础敌人。

教学内容：

- 地图目标。
- 采集节点。
- 基础攻击。
- 生命或防护反馈。
- 背包获得物品。

避免：

- 同时出现污染、防护模块、天气、地基、管线等复杂系统。
- 敌人难度过高。

### 20 到 30 分钟：第一次回基地加工

玩家体验：

- 返回基地。
- 把晶体矿物放入加工设备或库存。
- 启动基础配方。
- 获得第一个基地制造产物。

教学内容：

- 配方输入输出。
- 处理进度。
- 制造完成反馈。
- 产物用途。

关键反馈：

- 产物直接指向下一目标，例如过滤模块、净化剂或扫描组件。

### 30 到 40 分钟：第一次采样和污染提示

玩家体验：

- 地图提示附近有异常污染边界。
- 玩家前往边界外围采样。
- 污染造成防护压力，但不立即致命。
- 玩家采样异常晶体并就近回收残留物，再带回基地分析。

教学内容：

- 污染危险。
- 采样。
- 防护消耗。
- 样本分析。
- 配方或区域提示解锁。

避免：

- 把污染做成纯扣血惩罚。
- 让玩家不知道为什么进入污染区会失败。

### 40 到 50 分钟：第一次建造前置

玩家体验：

- 需要放置或启用一个过滤 / 处理设备。
- 玩家发现目标地块不能直接建造。
- 清理、平整或铺设基础地基。
- 成功放置设备或扩展基地功能。

教学内容：

- 可建造 / 不可建造。
- 可破坏 / 可清理。
- 地面不平或缺少地基的提示。
- 基础地板 / 地基。

关键反馈：

- 建造失败原因必须明确。
- 玩家修正问题后能立即成功建造。

### 50 到 60 分钟：第一次危险区域推进

玩家体验：

- 装备过滤模块或携带药剂。
- 进入污染边界更深处。
- 遭遇受扰生态或精英节点。
- 使用补给或特殊产物通过挑战。
- 解锁下一入口或获得遗迹异常提示。

教学内容：

- 装备 / 药剂 / 模块改变通行能力。
- 战斗准备有价值。
- 污染区提供更高价值样本。
- 主线悬念。

首小时结束反馈：

- 明确提示“更深区域已标记”。
- 回基地后显示新配方、新设备或新目标。

## 系统解锁顺序

建议顺序：

1. 移动和交互。
2. 基地库存。
3. 设备缺料。
4. 地图目标。
5. 采集。
6. 基础战斗。
7. 回基地加工。
8. 第二次外勤回收残骸材料。
9. 设备校准或修复型中间加工。
10. 制造远征产物。
11. 补给准备和小型战斗清障。
12. 污染和防护压力。
13. 采样和分析。
14. 清理、平整和地基。
15. 装备或使用模块 / 药剂。
16. 危险区域挑战。
17. 后续入口解锁。
18. 遗迹外圈受阻与回基地再准备。
19. 稳相信标改变再次深入结果。
20. 外圈中继确认。

首小时和紧随其后的第二闭环到这里即可收束。更深区域、前线回传、相位井与后续断面属于首小时之后的延伸内容，不再在本文按包逐项展开；具体进度和开发基线以 `docs/planning/current.md`、`docs/planning/vertical-slice.md`、`docs/design/development-retest-baselines.md` 和最新周志为准。
