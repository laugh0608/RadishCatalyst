# Core Gameplay Loop

更新时间：2026-04-25

## 文档目的

这份文档用于把项目上位方向转成后续 MVP、原型和系统设计都能引用的核心循环。

它不是功能清单，也不规定具体数值。它只回答：

- 玩家为什么反复出门。
- 玩家为什么反复回基地。
- 化工基地、探索战斗、角色成长和区域解锁如何互相推动。
- 哪些循环关系是首版必须守住的。

## 核心循环一句话

**基地准备远征，远征带回样本和资源，基地把它们转化为装备、工艺和通行能力，再推动玩家进入更危险的新区域。**

这条循环必须比单独的“建工厂”或“刷怪掉材料”更强。

## 主循环

1. 基地准备  
   玩家整理库存、制造补给、检查装备、处理污染、确认设备状态。

2. 选择目标  
   玩家根据当前任务、资源缺口、区域危险和基地需求，决定去哪里探索。

3. 外出探索  
   玩家进入野外、晶体矿脉、污染区、遗迹入口或秘境节点，进行采集、扫描、战斗和调查。

4. 风险应对  
   玩家处理怪物、环境危险、污染、氧气或防护余量、背包容量和撤离路线。

5. 带回成果  
   玩家带回原料、流体样本、生物材料、蓝图、设备核心、遗迹线索或污染残留。

6. 基地加工  
   基地通过反应、分离、储运、供能和污染处理，把成果转化为可用材料、装备、药剂、模块或新工艺。

7. 成长解锁  
   玩家获得更强装备、更稳定生产线、更好的环境适应能力和新的区域通行条件。

8. 进入下一层危险  
   新区域带来更高价值资源、更强生态反击、更复杂污染和更关键的主线线索。

## 三个互锁子循环

### 冒险循环

目标：

- 探路。
- 采样。
- 战斗。
- 回收。
- 撤离。

玩家在冒险循环中要持续判断：

- 继续深入还是返回。
- 采资源还是打敌人。
- 消耗补给还是保留撤离余量。
- 先调查主线还是先补基地短板。

### 基地循环

目标：

- 加工。
- 储运。
- 供能。
- 排障。
- 治理污染。

基地循环的重点不是让玩家一直盯着后台数字，而是让玩家感到基地是下一次远征的后勤核心。

基地应该回答：

- 这次带回来的东西能做什么。
- 下一次远征缺什么。
- 哪条生产链是瓶颈。
- 哪个污染或事故风险正在积累。

### 成长循环

目标：

- 角色更强。
- 装备更适配环境。
- 基地更稳定。
- 区域推进更深。

成长不应只来自打怪经验，也应来自：

- 工艺解锁。
- 样本分析。
- 设备升级。
- 防护和药剂改进。
- 新区域勘探。

## 世界压力循环

为了避免基地成为无风险后台，世界应对玩家行为有反馈：

- 基地污染升高，会改变附近生态和敌袭压力。
- 高价值资源开采，会引发更强的区域反应。
- 设备事故，会制造短期危机或任务后果。
- 秘境调查，会推动主线并打开更危险的路径。

这类压力应该让玩家重视污染处理、安全设施和远征准备，但不能让系统变成惩罚过重的维护游戏。

## 首颗星球的推荐节奏

首颗星球应承担教学和定调作用。

推荐节奏：

1. 建立基础前哨，恢复能源和反应器。
2. 扫描附近晶体矿脉，建立第一条稳定资源链。
3. 发现污染区，建设污染处理设施。
4. 进入污染边界采样，遭遇受扰生态。
5. 用样本解锁过滤模块、抗性补给或特殊加工。
6. 推进到遗迹入口或高价值秘境节点。

这一节奏不等同于 MVP 清单，只是后续拆功能时的方向参照。

## 多星球扩展循环

后续如果引入不同星球，每颗星球都应复用同一套主循环，但改变问题类型：

- 腐蚀星球：重点是耐腐蚀材料、酸雾治理和泄漏风险。
- 赤铁荒漠星球：重点是高温、粉尘、散热和矿物开采。
- 冰封挥发物星球：重点是低温流体、保温、冻结管线和储罐安全。
- 生物催化星球：重点是样本隔离、孢子污染、生物反应和生态反击。
- 遗迹区域：重点是异常能量、古代工艺核心、秘境战斗和主线真相。

## 设计边界

- 不把基地做成纯后台生产表。
- 不把探索做成单纯打怪掉材料。
- 不让化工复杂度变成专业考试。
- 不让战斗系统脱离基地后勤。
- 不让联机成为首版循环成立的前提。

## 对后续文档的要求

后续 `mvp-feature-list.md`、`resources-and-process-chain.md`、`maps-and-regions.md` 和角色成长文档，都应能指回这条循环。

如果某个功能无法说明它如何强化“基地准备 -> 外出探索 -> 带回成果 -> 基地加工 -> 成长解锁 -> 深入危险区域”，就应谨慎进入首版范围。
