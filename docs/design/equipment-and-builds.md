# Character Progression And Equipment - 装备与 Build

返回：[Character Progression And Equipment](character-progression-and-equipment.md)

## 装备结构

### 武器

武器负责战斗风格。

推荐方向：

- 工程步枪或能量步枪：稳定输出，适合通用探索。
- 近程切割器或冲击工具：高风险近战，兼具破障或采集用途。
- 投射器或发射器：使用特殊弹药，适合处理群体、装甲或污染体。
- 工具型武器：兼具扫描、采集、净化或打断效果。

武器升级应消耗基地加工产物，而不是只靠掉落同名武器。

### 防护服和装甲

防护服是连接角色成长和区域通行的核心装备。

它应承担：

- 基础防御。
- 环境过滤。
- 抗腐蚀、抗毒、抗寒、抗热或抗孢子。
- 能量供应和模块插槽。
- 事故或濒危状态下的短时保护。

高危区域不应只用等级墙阻挡，也可以要求玩家具备对应防护模块、过滤耗材或稳定剂。

### 模块

模块负责 Build 取舍。

推荐模块类型：

- 防护模块：提高特定环境抗性。
- 战斗模块：改变武器、技能或弹药效果。
- 探测模块：强化扫描、资源标记或弱点识别。
- 采集模块：提高采样、破障、回收或携带效率。
- 生存模块：改善能量、治疗、撤离或背包管理。

模块不宜一开始做成海量词条。首版更适合少量明确模块，每个模块都能改变远征选择。

### 消耗品

消耗品是基地生产反哺冒险的高频接口。

推荐类型：

- 治疗和修复包。
- 抗性药剂。
- 过滤罐、净化剂或稳定剂。
- 特殊弹药。
- 临时探测器。
- 便携能源、冷却剂或应急密封剂。

消耗品应让基地准备阶段有意义，但不能让玩家每次出门都陷入繁琐补货。

## 技能方向

技能应体现前哨工程战斗员身份，而不是传统魔法职业。

推荐技能类别：

- 武器技能：射击、切割、爆破、击退或弱点打击。
- 工程技能：部署临时装置、修复、过载、短时护盾或诱导敌人。
- 化工技能：投放净化剂、腐蚀弹、冷却剂、催化剂或稳定场。
- 探测技能：扫描资源、标记敌人弱点、揭示污染边界或发现遗迹线索。
- 生存技能：短时抗性、快速撤离、应急治疗或环境适应。

技能消耗可以使用能量、冷却、弹药、药剂或模块耐久。不同消耗类型应和基地后勤形成关系。

## Build 取舍

Build 的目标不是让玩家只追求最高伤害，而是围绕远征目标做准备。

推荐 Build 维度：

- 战斗型：更高输出和控场，适合清理敌巢、Boss 或秘境战斗。
- 探索型：更强扫描、背包、移动和采集效率，适合远距离勘探。
- 防护型：更强环境抗性和容错，适合污染区、腐蚀区或未知区域。
- 工程型：更依赖部署物、工具和特殊弹药，适合处理事故、装置和区域目标。

这些方向可以混搭，不应过早锁死职业。首版可以使用装备、模块和少量技能点形成轻量 Build，而不是完整职业体系。

## 与基地生产的咬合

角色成长和基地生产至少应形成以下关系：

- 武器升级需要金属、晶体、催化材料或遗迹部件。
- 防护升级需要过滤材料、抗性材料、稳定剂或污染处理副产物。
- 特殊弹药需要流体、中间体、能量晶体或生物样本。
- 药剂和临时抗性来自样本分析、过滤和生物处理。
- 探测器和扫描模块需要电子零件、晶体材料和区域样本。
- 区域通行能力需要基地解锁的装备模块或消耗品。

这意味着基地不是单纯建筑升级树，而是角色 Build 的供应链。

## 与地图和区域的咬合

不同区域应要求不同准备，而不是只提高敌人血量。

示例：

- 晶体矿脉区：需要采集工具、扫描模块和中等战斗能力。
- 污染边界区：需要过滤模块、净化剂和污染体应对能力。
- 腐蚀区域：需要抗腐蚀防护、密封剂和耐腐蚀装备材料。
- 生物催化区域：需要孢子过滤、样本隔离和群体敌人处理能力。
- 遗迹或秘境：需要探测模块、稳定剂、强战斗 Build 和充足补给。

区域要求应通过任务、地图图层、装备提示和危险反馈提前告知玩家，避免玩家在不知情的情况下被硬性惩罚。

## 掉落与制造边界

推荐原则：

- 怪物和秘境主要掉落样本、部件、蓝图、核心和特殊材料。
- 完整装备可以少量出现，但不应取代基地制造。
- 高级装备应由掉落材料和基地工艺共同完成。
- 随机词条可以远期评估，首版不应依赖复杂刷词条作为核心乐趣。

这样可以保留 ARPG 奖励感，同时不削弱化工基地的核心地位。

## 首版推荐范围

首版或 MVP 前后建议只验证最小闭环：

1. 两到三种武器或工具风格。
2. 一套防护服或装甲升级线。
3. 少量清晰模块，例如过滤、扫描、采集、战斗增强。
4. 一批基地制造的消耗品，例如治疗、抗性、特殊弹药。
5. 一个区域通行门槛，由防护模块或消耗品解决。
6. 一次秘境或高危区域挑战，验证装备、药剂、战斗和基地后勤的整体关系。

暂不建议首版就做：

- 多职业完整转职。
- 大型天赋树。
- 大量随机词条装备。
- 套装收集。
- 复杂宠物或随从养成。
- 以刷装为主的赛季化终局。
