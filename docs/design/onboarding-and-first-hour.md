# Onboarding And First Hour

更新时间：2026-05-10

## 文档目的

这份文档用于定义 RadishCatalyst 首次游玩的前 30 到 60 分钟体验。

它不是完整教程脚本，也不是 UI 文案表。它只回答：

- 玩家第一次进入游戏时按什么顺序学习系统。
- 哪些概念必须先学，哪些延后。
- 如何避免玩家被化工、基地、战斗、地形和污染同时压垮。
- 首小时结束时玩家应理解什么。

## 首小时目标

首小时结束时，玩家应该理解：

- 我是谁：前哨工程战斗员。
- 我在哪：异常星球上的受损化工前哨。
- 我为什么出门：基地缺资源和样本。
- 我为什么回基地：资源需要加工，装备和补给来自基地。
- 我为什么要处理污染：污染影响探索、敌人和建造。
- 我为什么变强：基地工艺产出了装备、药剂、模块或通行能力。

如果玩家只记住一句话，应是：

**基地让我能走得更远，远征让我能把基地建得更强。**

## 教学原则

- 一次只教一个新系统。
- 每个新系统必须立刻让玩家获得反馈。
- 先做动作，再解释原因。
- 用任务目标引导，不用大段说明。
- 教程文本要短，优先使用 UI 高亮、地图标记和设备状态。
- 玩家失败后应知道如何补救。
- 当主线开始重复往返更深区域时，应优先压缩无决策空跑，而不是让玩家为“基地有价值”额外支付纯移动时间。
- 不在第一小时展示完整复杂度。

## 首小时节奏

### 0 到 5 分钟：醒来或抵达前哨

玩家体验：

- 看见基地平台。
- 获得移动控制。
- 接收到前哨受损提示。
- 看到一个明确目标：检查前哨核心。

教学内容：

- 移动。
- 镜头。
- 交互键。
- 任务目标显示。

避免：

- 立即打开完整基地 UI。
- 立即介绍全部资源和配方。
- 大段世界观说明。

### 5 到 10 分钟：恢复基础设备

玩家体验：

- 打开基础库存。
- 查看一个设备缺料或停机。
- 明确需要外出找晶体矿物。

教学内容：

- 库存。
- 设备状态。
- 缺料提示。
- 基地是后勤核心。

关键反馈：

- 设备面板显示缺少输入。
- 地图标出晶体矿脉方向。

### 10 到 20 分钟：第一次外出采集

玩家体验：

- 离开基地平台。
- 到达晶体矿脉区。
- 使用工具采集资源。
- 遭遇少量基础敌人。

教学内容：

- 地图目标。
- 采集节点。
- 基础攻击。
- 生命或防护反馈。
- 背包获得物品。

避免：

- 同时出现污染、防护模块、天气、地基、管线等复杂系统。
- 敌人难度过高。

### 20 到 30 分钟：第一次回基地加工

玩家体验：

- 返回基地。
- 把晶体矿物放入加工设备或库存。
- 启动基础配方。
- 获得第一个基地制造产物。

教学内容：

- 配方输入输出。
- 处理进度。
- 制造完成反馈。
- 产物用途。

关键反馈：

- 产物直接指向下一目标，例如过滤模块、净化剂或扫描组件。

### 30 到 40 分钟：第一次采样和污染提示

玩家体验：

- 地图提示附近有异常污染边界。
- 玩家前往边界外围采样。
- 污染造成防护压力，但不立即致命。
- 玩家采样异常晶体并就近回收残留物，再带回基地分析。

教学内容：

- 污染危险。
- 采样。
- 防护消耗。
- 样本分析。
- 配方或区域提示解锁。

避免：

- 把污染做成纯扣血惩罚。
- 让玩家不知道为什么进入污染区会失败。

### 40 到 50 分钟：第一次建造前置

玩家体验：

- 需要放置或启用一个过滤 / 处理设备。
- 玩家发现目标地块不能直接建造。
- 清理、平整或铺设基础地基。
- 成功放置设备或扩展基地功能。

教学内容：

- 可建造 / 不可建造。
- 可破坏 / 可清理。
- 地面不平或缺少地基的提示。
- 基础地板 / 地基。

关键反馈：

- 建造失败原因必须明确。
- 玩家修正问题后能立即成功建造。

### 50 到 60 分钟：第一次危险区域推进

玩家体验：

- 装备过滤模块或携带药剂。
- 进入污染边界更深处。
- 遭遇受扰生态或精英节点。
- 使用补给或特殊产物通过挑战。
- 解锁下一入口或获得遗迹异常提示。

教学内容：

- 装备 / 药剂 / 模块改变通行能力。
- 战斗准备有价值。
- 污染区提供更高价值样本。
- 主线悬念。

首小时结束反馈：

- 明确提示“更深区域已标记”。
- 回基地后显示新配方、新设备或新目标。

## 系统解锁顺序

建议顺序：

1. 移动和交互。
2. 基地库存。
3. 设备缺料。
4. 地图目标。
5. 采集。
6. 基础战斗。
7. 回基地加工。
8. 第二次外勤回收残骸材料。
9. 设备校准或修复型中间加工。
10. 制造远征产物。
11. 补给准备和小型战斗清障。
12. 污染和防护压力。
13. 采样和分析。
14. 清理、平整和地基。
15. 装备或使用模块 / 药剂。
16. 危险区域挑战。
17. 后续入口解锁。
18. 遗迹外圈受阻与回基地再准备。
19. 稳相信标改变再次深入结果。
20. 外圈中继确认。
21. 深段回波回收。
22. 基地解析更深坐标。
23. 把更深坐标写回深段门禁。
24. 深段入口回收相位纤丝。
25. 污染过滤器精炼谐振滤芯。
26. 基础反应器组装深段覆写栓。
27. 返回深段入口覆写锁扣并带回样块。
28. 回基地解析深段样块，整理路由印片。
29. 返回深段点亮阵列台，清理追袭体并回收相位导管。
30. 把相位导管带回基地整理成深段读数矩阵。
31. 当深段往返开始变成长距离空跑时，部署基地控制的前线回传锚点，缩短无决策往返但保留回基地加工价值。
32. 在深段按锚点快速回基地，并在基地相位回投台返回最后部署锚点。
33. 从回投台重返更东侧裂相脊，击败裂相猎手并回收裂相碎屑。
34. 回处理点污染过滤器，把裂相碎屑筛成透镜胚片并保留副产污染浆液。
35. 回基地基础反应器，把透镜胚片、污染浆液和基础零件调准成中继调谐镜。
36. 带着中继调谐镜返回裂相尖塔，校准后带回第一份内层故障轨迹。

不应在首小时同时解锁：

- 完整管线。
- 多设备自动化。
- 多天气。
- 多角色 Build。
- 大型任务日志。
- 复杂技能树。
- 联机入口。

## 开发复测基线

为了避免首小时内容扩展后每次人工复测都从空档重打，当前原型已把这条链拆成 `S0` 到 `S5` 六个开发基线。

- `S0`：新档起点，保留完整冷启动链。
- `S1`：处理点已就绪，直接进入污染边界段。
- `S2`：外圈中继已确认，直接进入外圈回波与更深坐标链。
- `S3`：深段门禁已开，直接进入相位纤丝与覆写栓链。
- `S4`：深段样块已回收，直接进入第二轮阵列解析链。
- `S5`：前线回传已上线，直接观察回投后的裂相尖塔链与深段往返节奏。

这套基线是开发内部工具，不是正式玩家存档流程。详细设计与使用方式见 `docs/design/development-retest-baselines.md`。

## UI 引导

首小时 UI 应优先包含：

- 当前目标。
- 生命、防护和快捷栏。
- 小地图或方向标记。
- 设备缺料提示。
- 配方输入输出。
- 建造失败原因。
- 污染危险提示。
- 返回基地提示。
- 常驻 HUD 不遮挡关键推进区域。

可延后：

- 完整生产统计。
- 复杂地图图层。
- 全资源图鉴。
- Wiki 入口。
- 官方工具入口。

## 原型落地顺序

首个 Godot 原型的引导实现应先做“能完成目标”的最小链路，不先做完整教程系统。

优先实现：

1. 当前目标文本。已在原型 HUD 中显示目标、进度和阶段短提示。
2. 地图或方向标记。已用临时地图色块、区域文字、正式小地图 / 地图标记雏形表达当前目标方向与关键区域状态。
3. 交互提示。已在靠近交互对象时显示提示。
4. 设备缺料提示。已在靠近加工设备时显示当前配方输入、产出、加工进度和缺料状态，并可用 `Q` 打开设备面板雏形。
5. 建造失败原因。已接入最小清理、地基和污染过滤器建造失败提示。
6. 污染危险提示。已在污染采集、当前区域污染状态、低防护进入污染边界和撤离补救中起步。
7. 回基地提示。已在样本返回、撤离补救和方向提示中起步。

暂缓实现：

- 完整教程弹窗框架。
- 大量语音或剧情演出。
- 可回看的教学百科。
- 完整键位自定义引导。

只有当玩家能不依赖开发者说明完成第一条闭环后，再扩展教程表现。

当前原型中，玩家已经能看到目标、目标进度、小地图 / 区域标记、任务驱动短提示、生命、防护、当前区域污染状态、当前模块、快捷栏、关键物资摘要、交互提示、交互日志、补给使用短反馈、任务完成反馈和撤离结果反馈。常驻 HUD 已改成分散式结构：左上区域地图、左侧目标卡、右上生存 / 污染状态卡和左下提示 / 日志分离显示，不再把主线信息强塞进单列；区域方向、设备详情、长提示和运行反馈分别交给小地图、设备面板、交互提示、任务完成面板、撤离面板和日志承担。世界相机也已锁定玩家跟随，外圈和深段推进不会再因为静态舞台摆位而丢出视野。

实现边界上，上述 HUD 信息已经拆到独立 presenter：状态摘要由 `HudStatusPresenter` 生成，小地图 / 区域标记由 `HudMapPresenter` 生成，任务短提示由 `HudHintPresenter` 生成，设备面板由 `HudDevicePanelPresenter` 生成，任务完成 / 撤离 / 补给反馈由 `HudFeedbackPresenter` 生成，日志格式化由 `HudLogPresenter` 生成。后续若继续调整首小时提示，应优先改对应 presenter 和流程检查，不把文案重新堆回 `PrototypeHud` 或 `GameRoot`。

区域标记会显示基地、晶体、污染和遗迹四个关键区域的相对方向、解锁状态与当前目标。污染状态会显示当前区域是否稳定、污染强度、过滤模块是否生效、防护消耗倍率和低防护建议。靠近基础反应器或污染过滤器时，HUD 交互提示和设备面板会显示设备名、当前配方输入、产出、副产、耗时、加工进度、当前目标推荐配方、最近完成配方、产物 / 副产入库去向、下一步建议和可加工 / 缺料 / 未解锁状态；加工完成日志也会说明产物进入背包以及后续行动。

靠近粗糙地块、基础地基建造点或污染过滤器建造点时，HUD 会显示地块是否阻挡建造、清理后续、建造材料、基础地基数量、缺料或前置原因，以及可建造时的 `E` 键提示。交互、攻击、配方切换、加工、建造和区域门槛失败时，日志会按失败类型、原因和下一步建议提示补救方向。

当前目标已经能随恢复前哨、进入晶体矿脉区、采集晶体、回收外勤残骸、回基地组装反应器校准件、采样异常晶体、就近回收异常残留物、回基地分析异常样本、制造过滤模块、调制修复凝胶、清理处理点北缘任务阶段敌人、扩建处理点、启用模块、进入污染边界、采集污染沉积物、处理抗污染药剂、击败污染敌人、压制污染残核、确认封锁遗迹入口、进入遗迹外圈、回收继电残片、回基地组装稳相信标、稳定抖动雾幕、确认外圈中继、清理相位守卫、回收外圈回波匣、回基地解析深段回波、把更深坐标写回门禁、进入深段入口、回收相位纤丝、回基地精炼谐振滤芯、组装深段覆写栓并返回深段锁扣推进、解析深段样块、点亮深段阵列、整理深段读数矩阵，并在深段固定点部署前线回传锚点；任务完成反馈会按完成、奖励、解锁、提示和新目标的顺序显示摘要。前哨核心恢复、晶体簇采集、外勤残骸回收、异常晶体采样、异常残留点回收、粗糙地块清理、污染沉积斑采集、受扰掠行体击败、封锁遗迹入口确认、继电残片回收、抖动雾幕稳定、外圈中继确认、外圈回波匣回收、深段门禁写入、相位纤丝回收、深段锁扣覆写和前线回传锚点部署都会在场景内保留完成态，避免玩家只能从日志判断交互是否生效。

深段读数矩阵完成后，深段固定点会开放前线回传锚点，基地平台也会开放相位回投台；玩家可先在前线按锚点快速回基地，再从基地回投到最后部署锚点，压缩纯空跑，但不跳过基地加工、补给和再准备。

真正用过一次回投台后，更东侧裂相脊会暴露新的裂相猎手、裂相碎屑和裂相尖塔；玩家需要先把裂相碎屑带回处理点筛成透镜胚片，再回基地把胚片和污染浆液调准成中继调谐镜，最后把这份基地产物重新带回前线校准裂相尖塔，带回第一份内层故障轨迹。

晶体矿脉区会要求先恢复前哨核心，污染边界深处会要求先扩建处理点并启用基础过滤模块，遗迹外圈深段会要求先回基地组装稳相信标并稳定抖动雾幕，外圈中继之后会要求继续清理相位守卫并把回波匣带回基地分析，深段门禁则要求先把 `deep_ruin_coordinates` 真正写回现场，深段锁扣还会要求先回基地用污染过滤器和基础反应器组装深段覆写栓；被挡回时，日志会补充处理点扩建、启用过滤模块、补给防护、回基地准备稳相信标、回基地解析深段回波或回基地精炼相位纤丝 / 组装覆写栓等具体原因。生命或防护耗尽会自动撤回前哨，并通过 HUD 撤离反馈面板提示撤离原因、恢复结果，以及使用修复凝胶、抗污染药剂、启用过滤模块或回基地补给的再尝试建议。完成污染边界目标后会激活污染残核压制，压制后地图东侧封锁遗迹入口会转为遗迹外圈入口；确认外圈中继后 `slice_01_complete` 会先解锁以保留第一切片终点，后续再通过外圈回波匣、基地分析、深段门禁、相位纤丝精炼和深段覆写栓，把“更深入口”变成实际可进入的新区域和新收益。

玩家可用 `1` 使用修复凝胶、用 `2` 使用抗污染药剂，主动补充生命或防护；处理点补给准备段会先要求调制修复凝胶，再进入清障战斗，让玩家看到“回基地加工补给”对下一次外勤的价值。恢复前哨后，前哨核心本身也会提供基地整备：回基地可直接恢复生命和防护，保证旧图材料耗尽后仍能继续验证更深推进，而修复凝胶的价值继续保留在前线续战。污染边界段会提示抗污染药剂来自过滤器处理沉积物，使用后反馈会指向继续深入污染边界或压制污染残核。使用成功时会显示恢复量、当前生命 / 防护和剩余数量，满状态、缺少物品或空槽时会显示保留补给、回基地加工或重新绑定的短提示。HUD 左侧调试面板可把两个快捷栏槽位切换为修复凝胶、抗污染药剂 I 或空槽；三槽位存档调试面板支持保存、读取、清空和从新档重置，`K` / `L` 仍保留为默认槽位快捷键。

仍未接入：

- 更完整的建造面板、建造预览和正式地块状态表现。
- 更正式的污染危险 UI、返回基地提示表现和死亡 / 撤离演出；当前只有面板雏形。
- 更完整的补给面板、物品冷却和正式消耗品表现；当前只有快捷栏与短反馈。
- 正式存档 / 快捷栏管理界面；当前仍是原型调试面板。
- 完整教程弹窗或可回看教学。

## 失败与补救

首小时允许玩家失败，但必须可恢复。

推荐规则：

- 战斗失败后回到基地或撤离点。
- 关键任务资源不要永久丢失。
- 防护不足时给出明确提示。
- 背包满时提示返回基地。
- 建造失败时说明原因和解决方式。
- 污染区失败后提示需要过滤模块、药剂或更多补给。

不推荐：

- 第一次死亡损失大量进度。
- 让玩家因为没读文本而卡关。
- 天气、污染或敌人随机阻断主线。

## 文案和提示风格

提示应短而具体。

推荐：

- “设备缺少晶体矿物。”
- “污染浓度升高，防护消耗加快。”
- “地面不平，先平整或铺设地基。”
- “过滤模块可降低污染区防护消耗。”
- “样本分析完成：污染边界已标记。”

避免：

- 长段教程说明。
- 专业公式。
- 含糊提示，例如“条件不足”。
- 只用颜色表达危险。

## 首小时验收标准

首小时体验可用时，应满足：

- 玩家能在无开发者口头解释下完成第一闭环。
- 玩家知道为什么要回基地。
- 玩家至少制造一次能改变外勤结果的产物。
- 玩家理解污染、防护和区域推进的关系。
- 玩家理解地块为什么不能建造，以及如何修正。
- 玩家完成一次小型战斗挑战。
- 玩家看到后续区域或主线悬念。

## 当前阶段结论

RadishCatalyst 的首小时不是把所有系统展示给玩家，而是让玩家完成一次可信闭环。

首小时必须保护一件事：

**玩家先理解“基地和外勤互相需要”，再逐步理解化工、污染、地形、天气和长期成长。**
