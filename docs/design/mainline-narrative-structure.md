# Narrative And Quest Framework - 主线叙事结构

返回：[Narrative And Quest Framework](narrative-and-quest-framework.md)

## 文档目的

这份文档用于定义 RadishCatalyst 首版叙事、主线任务、支线任务和教程任务的组织方式。

它不是完整剧本，也不是任务表。它只回答：

- 首版故事为什么开始。
- 主线如何服务核心循环。
- 支线任务承担什么功能。
- 教程任务如何自然融入世界。
- 叙事如何避免拖累原型开发。

## 叙事定位

首版叙事应是“工业现场任务记录 + 异星异常调查”，而不是长篇剧情演出。

玩家应通过行动理解世界：

- 修复前哨。
- 采集资源。
- 分析样本。
- 处理污染。
- 遭遇受扰生态。
- 解锁更危险区域。

文本应短、明确、带工程语气。首版不需要大量角色对白、复杂阵营或多分支剧情。

## 首版主线一句话

玩家抵达或苏醒在一座受损的异星化工前哨中；为了恢复生存能力并调查周边异常污染，玩家必须外出采集晶体资源和异常样本，重启基础加工链，制造防护与过滤产物，最终进入污染边界并发现遗迹或深层异常入口。

## 主线阶段

### 阶段一：恢复前哨

目的：

- 让玩家认识基地是安全区和后勤核心。
- 交代前哨受损、能源不足、加工链中断。
- 引导玩家恢复基础库存和设备。

任务目标示例：

- 检查前哨核心。
- 打开基础库存。
- 重启基础加工设备。
- 查看当前缺失资源。

系统教学：

- 基地 UI。
- 库存。
- 设备状态。
- 当前目标。

### 阶段二：勘探晶体矿脉

目的：

- 让玩家第一次离开基地。
- 让资源采集、战斗和撤离成为一组动作。
- 说明晶体矿物是基础工艺链的关键。

任务目标示例：

- 前往晶体矿脉区。
- 扫描晶体节点。
- 采集晶体矿物。
- 击退原生小型生物。
- 带回资源。

系统教学：

- 移动。
- 采集。
- 基础战斗。
- 背包和撤离。

### 阶段三：分析异常样本

目的：

- 把采样与主线调查关联起来。
- 让玩家知道样本不是普通材料。
- 打开污染边界和受扰生态线索。

任务目标示例：

- 采集异常晶体碎片。
- 回基地分析样本。
- 解锁污染残留处理配方。
- 标记污染边界区。

系统教学：

- 采样。
- 样本分析。
- 配方解锁。
- 地图标记。

### 阶段四：建立污染处理能力

目的：

- 让污染从背景变成系统。
- 让基地生产直接反哺外勤。
- 引入地基、平整和建造前置。

任务目标示例：

- 清理一处可建造地块。
- 平整地面或铺设基础地基。
- 建造或启用过滤 / 处理设备。
- 制造过滤模块、净化剂或抗性药剂。

系统教学：

- 可建造 / 不可建造。
- 清理和平整。
- 地基。
- 制造远征产物。

### 阶段五：进入污染边界

目的：

- 验证玩家是否理解装备、药剂和模块准备。
- 让污染区成为第一个危险层级提升。
- 通过受扰生态和污染源点承接战斗压力。

任务目标示例：

- 装备过滤模块。
- 携带抗性药剂或净化剂。
- 进入污染边界。
- 采集污染沉积物。
- 击败受扰污染体。

系统教学：

- 环境危险。
- 防护消耗。
- 快捷物品。
- 敌人掉落样本。

### 阶段六：治理污染源点

目的：

- 给切片一个小型高潮。
- 让战斗、补给、污染治理和样本分析合并。
- 打开后续区域悬念。

任务目标示例：

- 定位污染源点。
- 清理周边敌人。
- 使用净化剂或稳定剂。
- 击败精英节点。
- 回收关键样本。
- 解锁遗迹入口或深层污染读数。

系统教学：

- 精英挑战。
- 区域目标。
- 解锁新入口。
- 切片结束反馈。

## 当前原型主线延伸结构

截至 2026-05-14，当前 Godot 原型的主线已经不再停在“进入污染边界并发现遗迹入口”，而是沿着单张向东切片继续验证了多轮“基地整备 -> 更深外勤 -> 新区域解锁”的结构，并在前线回传锚点之后开始刻意打破同构模板。

当前已验证的主线收益锚点依次是：

```text
外圈回波匣 -> 深段样块 -> 深段读数矩阵 -> 内层故障轨迹
-> 相位井定位器 -> 相位井芯样本 -> 相位井心核
-> 相位井纺核 -> 相位井织核 -> 相位井结核
-> 相位井锚核 -> 相位井余响片 -> 相位井稳窗读数
```

前线回传锚点之前，这些后续任务包大多保持统一：

1. 先把上一跳带回的关键样本或遗迹件带回基地解析。
2. 再沿更东侧推进，击败一个新敌人并回收两处关键资源。
3. 回基地或处理点完成必要加工。
4. 带着新产物返回更东侧终点，勘验后拿到下一跳推进锚点。

前线回传锚点之后，当前主线已经开始把“单件门禁制造”改成整备包任务形态：

1. 带回一组现场材料或样本。
2. 回基地解析上一跳成果，回处理点稳定现场材料。
3. 在同一个整备任务内完成过滤中间件和装配件。
4. 下一趟外勤直接进入现场校准、开锁、勘验或短守场，不再插入一个独立组装任务。

相位井锚核之后，锚场回稳进一步验证了另一种后段任务形态：

1. 回基地把锚核解析成归谱片和锚核落尘。
2. 完成锚场整备，把锚核落尘稳定成稳场滤囊，并组装井系校锚桩。
3. 带着基地产物返回井系桥东侧，先部署，再压住井系守脉体，最后完成回稳交互。
4. 把局部稳定窗口和相位井余响片作为现场收益，再回基地解析成相位井稳窗读数，把叙事收益落成可回访的前线回充点。

这种结构让当前原型的叙事仍保持“工业现场任务记录 + 异星异常调查”口径，而不是突然切成大段剧情演出：

- 基地解析负责解释当前异常物为什么值得继续向东推进。
- 新敌人和新区域负责给风险升级一个可玩的现场原因。
- 整备包负责把“为什么回基地”解释成一次可执行准备，而不是只回去做一个门禁件。
- 新产物负责把收益落成下一次外勤许可或现场收益，而不是只落成一条文本说明。
- 前线回传锚点和基地相位回投台负责在晚段维持节奏，不让主线退回长距离空跑。
- 稳窗读数负责说明“这段前线已经被基地校准过”，因此它是完成态反馈，不是继续向东扩一段新门禁。

短期叙事约束保持不变：文本仍应短、明确、带工程语气；当前不需要长对白树、阵营分支或复杂角色演出，优先把“为什么回基地、为什么还能继续向东”解释清楚。
