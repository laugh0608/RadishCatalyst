# Official Tools

本目录用于规划和承载未来面向玩家的官方辅助工具。

这里的工具不是开发脚本；开发脚本仍放在仓库根目录 `tools/` 或 `scripts/` 中。

## 工具目标

官方工具用于帮助玩家在游戏外理解复杂系统、规划生产路线和分享探索结果。

候选工具包括：

- 在线合成台：查询某个物品的配方、上游材料、下游用途和替代路线。
- 工艺链规划器：从目标产物反推需要的设备、原料、产能和副产物处理。
- 地区资源标点：记录区域资源点、危险点、秘境入口和特殊事件。
- 装备 / 药剂计算器：比较材料投入、属性收益和适用流派。
- 怪物 / 掉落查询：按区域、危险等级和掉落物反查敌人。
- 蓝图 / 生产线分享：未来支持玩家导入导出生产线配置。

## 当前阶段原则

- 当前只做规划，不立即实现完整工具。
- 工具数据应优先来自结构化游戏数据，而不是手写复制 Wiki 页面。
- 工具 UI 应面向玩家，不能暴露内部开发状态或未公开剧情。
- 官方工具应保留离线静态部署可能性，优先考虑 Web 形态。
- 如果未来游戏支持 Mod，工具需要区分官方数据和玩家自定义数据。

## 推荐结构

- `docs/`：工具规划、信息架构、数据需求和 UI 草案。
- `web/`：未来 Web 工具前端。
- `shared-data/`：未来工具侧可消费的数据样例或导出格式说明。

## 与 Wiki 的关系

- Wiki 负责解释“是什么、在哪里、怎么用”。
- 官方工具负责回答“为了得到 X，我需要什么路线、设备和区域信息”。
- 两者应共享同一套数据源，避免玩家看到互相矛盾的信息。
