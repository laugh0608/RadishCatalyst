# 晶体矿物加工

更新时间：2026-05-14

## 配方

```text
晶体矿物 x3
-> 基础反应器
-> 基础零件 x4
```

## 定位

这条配方负责早期教学和应急补给，让玩家理解“外出采集 -> 回基地加工 -> 获得装配材料”的基础循环。

## 后段补给口径

当前后段主线不再只依赖有限晶体簇。完成关键前线阶段后，前哨核心会按回收 / 阶段补给口径返还 `基础零件`，用于支撑后续任务装配。首次晶体采集完成后，晶体矿脉区北侧还有可选 `富晶残脉` 兜底；相位中继锚点部署后，也可以用污染浆液回收基础零件。
