# 相位井锚场回稳生产线

更新时间：2026-05-13

剧透提示：本页涉及相位井后段生产链。

## 目标

把 `相位井锚核` 处理成可部署的 `井系校锚桩`，再把这件基地产物带回前线，换回 `相位井余响片`，并解析成可校准前线回充点的 `相位井稳窗读数`。

## 生产线总览

```text
相位井锚核 + 基础零件
-> 基础反应器
-> 相位井归谱片 + 锚核落尘

锚核落尘
-> 污染过滤器
-> 稳场滤囊 + 污染浆液

相位井归谱片 + 稳场滤囊 + 基础零件
-> 基础反应器
-> 井系校锚桩

井系校锚桩
-> 井系桥东侧锚场回稳窗
-> 相位井余响片

相位井余响片 + 基础零件
-> 基础反应器
-> 相位井稳窗读数
-> 锚场回稳窗可回访回充
```

## 设备分工

- `基础反应器`：负责解析锚核、组装 `井系校锚桩`，并在回稳完成后解析 `相位井余响片`。
- `污染过滤器`：负责把 `锚核落尘` 稳定成 `稳场滤囊`。

## 这条线的特殊点

- 它第一次把“解析配方的副产物”直接变成下一步主输入，而不是只做陪衬。
- 它的最终收益不是直接开门，而是先让玩家回前线完成一次部署和回稳，再把完成态校准成可回访的前线回充点。
- 它仍然只复用 `基础反应器` 和 `污染过滤器` 两台旧设备，没有额外引入第三台专用设备。
- 线内需要的基础零件不再只靠有限晶体簇支撑；关键前线阶段会按前哨核心回收 / 阶段补给口径返还一批基础零件。

## 常见问题

- 没看到 `稳场滤囊`：
  先确认 `锚核落尘` 已经从锚核解析里产出，再去污染过滤器处理。
- 带着 `相位井归谱片` 还是不能组装：
  还需要 `稳场滤囊` 和基础零件。
- 组装完 `井系校锚桩` 以后下一步是什么：
  回井系桥东侧，把它部署到锚场回稳窗。
- 拿到 `相位井余响片` 以后下一步是什么：
  回基地基础反应器解析成 `相位井稳窗读数`；之后锚场回稳窗可作为回访回充点使用。
