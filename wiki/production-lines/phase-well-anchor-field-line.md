# 相位井锚场回稳生产线

更新时间：2026-05-11

剧透提示：本页涉及相位井后段生产链。

## 目标

把 `相位井锚核` 处理成可部署的 `井系校锚桩`，再把这件基地产物带回前线，换回 `相位井余响片`。

## 生产线总览

```text
相位井锚核 + 基础零件
-> 基础反应器
-> 相位井归谱片 + 锚核落尘

锚核落尘
-> 污染过滤器
-> 稳场滤囊 + 污染浆液

相位井归谱片 + 稳场滤囊 + 基础零件
-> 基础反应器
-> 井系校锚桩

井系校锚桩
-> 井系桥东侧锚场回稳窗
-> 相位井余响片
```

## 设备分工

- `基础反应器`：负责解析锚核，并在最后组装 `井系校锚桩`。
- `污染过滤器`：负责把 `锚核落尘` 稳定成 `稳场滤囊`。

## 这条线的特殊点

- 它第一次把“解析配方的副产物”直接变成下一步主输入，而不是只做陪衬。
- 它的最终产物不是直接开门，而是把玩家送回前线完成一次部署和回稳。
- 它仍然只复用 `基础反应器` 和 `污染过滤器` 两台旧设备，没有额外引入第三台专用设备。

## 常见问题

- 没看到 `稳场滤囊`：
  先确认 `锚核落尘` 已经从锚核解析里产出，再去污染过滤器处理。
- 带着 `相位井归谱片` 还是不能组装：
  还需要 `稳场滤囊` 和基础零件。
- 组装完 `井系校锚桩` 以后下一步是什么：
  回井系桥东侧，把它部署到锚场回稳窗。
