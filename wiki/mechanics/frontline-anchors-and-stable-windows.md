# 前线锚点与稳定窗口

更新时间：2026-05-17

剧透提示：本页涉及深段后半与相位井后段机制。

## 前线锚点是什么

前线锚点是深段后半开放的前线落点。它的作用不是做成全图自由传送，而是缩短“前线回基地整备，再返回最近进度点”的纯跑图时间。

## 当前规则

- 已部署的前线锚点会保留，不会因为你后来部署了新的锚点而消失。
- 同一时间只会有 1 个“当前回投目标”。
- 基地相位回投台默认把你送到当前回投目标，而不是任意地点。
- 如果已经部署了多个锚点，可以在基地相位回投台按 `R` 切换当前落点。

## 稳定窗口是什么

稳定窗口是井系桥东侧回稳完成后的局部安全结果。它的意义有两层：

- 说明这段前线已经被压成可重复进入的稳定点。
- 把后段目标从“再捡两份材料”改成“先把基地产物带回现场，再完成一次回稳”。
- `相位井余响片` 解析成 `相位井稳窗读数` 后，这个窗口会进一步变成可回访的生命 / 防护回充点。
- 稳窗相位序校准完成后，这里还能承接轻量前线行动和第一轮基地行动选择，例如读取稳窗回波、补给回执、巡线信标或相位测绘点并带回基地解析。

## 井系校锚桩与回稳规则

- `井系校锚桩` 需要先通过锚场整备完成，才能带去现场部署。
- 部署后不会立刻完成回稳，必须先处理现场压力源。
- 当前压力源包括两处压力钉和 `井系守脉体`；先清压力钉，再压住守脉体，回稳交互才会真正完成。
- 失败后不需要回基地重做 `井系校锚桩`；已部署状态会保留，允许直接重试。
- 回稳完成后会先现场回充一次；解析出 `相位井稳窗读数` 后，后续回访也可在这里回充生命和防护。

## 轻量前线行动与基地选择

轻量前线行动不是完整远征系统。当前已经验证三轮短闭环，并在后续加入第一轮基地行动选择：

```text
基地确认前线行动
-> 回到井系桥东侧读取稳窗回波探点
-> 回基地基础反应器解析稳窗回波样本
-> 前线行动回报
-> 补给回执 / 巡线信标两条短行动
-> 稳场补给或相位测绘二选一
```

靠近基础反应器时，当前任务会自动推荐对应解析配方。`稳场补给` 是一处低风险投放点，返回收益偏整备资源；`相位测绘` 是两处前线测绘点，返回收益偏路线提示。当前这仍是最小选择原型，不包含完整行动队列或长期派遣。

## 当前设计边界

- 这套机制当前仍然只服务“基地服务冒险，冒险反哺基地”的主线闭环。
- 它不是完整波次防守系统。
- 它也不是完整自由传送网。
- 它也不是完整远征队列；补给 / 测绘只用于验证行动前取舍能否改变下一趟前线目标和返回收益。
- `相位井稳窗读数` 先把窗口变成可回访收益；后续相位序校准和前线行动只验证短回访模板，不代表已经进入完整大远征系统。
