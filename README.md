# RadishCatalyst

RadishCatalyst! 萝卜异星催化！

项目文档入口：[docs/README.md](docs/README.md)

## Repository Layout

- `docs/`：策划、设计、架构、参考资料与归档。
- `wiki/`：未来面向玩家的 Wiki 源内容。
- `official-tools/`：未来面向玩家的官方辅助工具。
- `client/`：未来 Godot 客户端工程。
- `server/`：未来可选服务端或联机实验工程。
- `assets/`：可提交的源资产与资产说明。
- `tools/`：项目脚本和自动化工具。
- `scripts/`：仓库检查与自动化脚本。
- `.github/`：PR 模板、GitHub Actions 和 ruleset 模板。
